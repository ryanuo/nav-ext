<h1 align="center">N-Tab</h1>

## 🧩 项目说明

- N-Tab是一个轻量级浏览器扩展的前端界面部分，采用 Vue 3 + TypeScript + Pinia 架构开发，主要功能包括时间显示、搜索建议、搜索引擎切换、快捷翻译、主题配置等。
